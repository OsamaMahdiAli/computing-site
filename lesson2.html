الدرس الثاني: التعاون (Collaboration) عبر Git

في هذا الدرس نشرح كيفية تعاون عدة مطورين على مشروع واحد باستخدام Git.

  • git clone — لنسخ المشروع.
  • git pull — لجلب التحديثات من GitHub.
  • branches — لإنشاء فروع للعمل بدون تأثير على النسخة الأساسية.
  • pull requests — لدمج التغييرات بين أعضاء الفريق.
